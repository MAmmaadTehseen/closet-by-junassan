Drop your Ceramic font file here as `Ceramic.otf`.

If you have multiple weights/styles, rename them like:
  Ceramic-Regular.otf
  Ceramic-Medium.otf
  Ceramic-Bold.otf
  Ceramic-Italic.otf

Then update src/app/layout.tsx localFont.src[] accordingly.
The default single-file config expects just Ceramic.otf (400 weight, normal style).
